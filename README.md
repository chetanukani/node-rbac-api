## Prerequsite

- NodeJS >= v16.18.3
- mongoDB >= v4.2.3

## Installation

1. Clone Repo

```
$ git clone url
$ cd node-rbac-api
```

2. create `config/.env` file
3. add credentials as `config/.env.example` file

4. install node modules

```
$ npm i
```

5. start web server

```
$ npm run dev
```

6. server will start on port define in `config/.env` file
